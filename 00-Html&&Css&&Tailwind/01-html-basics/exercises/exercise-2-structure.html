Exercise 2: Building Proper HTML Structure

Instructions

This exercise focuses on proper HTML structure and organization. Complete each section below.

Task 1: Create a Website Header

Create a header section for a personal website that includes:

  • Main site title (h1)
  • Navigation menu with 4 links (Home, About, Projects, Contact)
  • A welcome message paragraph

Task 2: Create Main Content Sections

Create three main content sections:

About Section

Include: section heading (h2), 2 paragraphs about yourself, and your photo

Skills Section

Include: section heading (h2), unordered list of skills you want to learn

Goals Section

Include: section heading (h2), ordered list of your learning goals
